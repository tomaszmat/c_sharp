Kernel Examples
Benchmark Example This example gathers and prints benchmark timing data for some representative SYS/BIOS APIs.
Clock Example The clock example shows how to use the ti.sysbios.knl.Clock module to create one-shot and periodic Clock Instances. Clock Instances are essentially functions that run after a certain number of Clock ticks.
Error Example This example shows how to use Error_Blocks to catch runtime errors. An Error_Block is passed to Task_create() and then checked for create failure. When no Error_Block is passed, the application will terminate when an error occurs. The error.cfg file shows how to plug an error hook.
Event Example The event example shows how to use the ti.sysbios.knl.Event module. The ti.sysbios.knl.Event module solves the 'Wait on Multiple' problem. In this example a reader task waits on several events.
Hello Example The hello example serves as a basic sanity check program for SYS/BIOS. It demonstrates how to print the string 'hello world' to stdout.
Memory Example This example shows the use of xdc.runtime.Memory module and different xdc.runtime.IHeap implementations to manage memory. A system heap is created statically using ti.sysbios.heaps.HeapMem.
Task Mutex Example This example shows the use of two tasks and one semaphore to protect data access.
Swi Example The swi example shows how to use the ti.sysbios.knl.Swi module. It demonstrates how the various Swi APIs work.

Follow these steps to create a project that uses TI-RTOS:

  1. First, choose Project > Create New Project from the IAR menus.
  2. Select the Tool chain, and choose the Empty project template. Click OK.
  3. Browse to the location where you want to save this project, and type a file name for the project file (*.ewp). Click Save. Each project should be saved in a separate directory.
  4. Click on an example listed above to add the files for that example to your project.
  5. Choose File > Save Workspace
  6. Browse to the location where you want to save this workspace, and type a file name for the workspace file (*.eww). Click Save.
  7. Choose Project > Make.
  8. Use the text editor in IAR to modify the .c and .cfg files in the example, and rebuild the project as needed.
  9. See the following sources for more information about using TI-RTOS.
    • TI-RTOS Getting Started Guide (/docs)
    • TI-RTOS User's Guide (/docs)
    • Creating TI-RTOS Applications in IAR Embedded Workbench page in the Texas Instruments Wiki
